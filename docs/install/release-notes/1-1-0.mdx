## Highlights

- Support added for 15 color themes during interactive mode
- New confif file `color_theme` option used to define selected theme
- `conjure list <any-string>` now provides proper error handling
- Updated to go 1.26.0

## Known Issues

- No support for remote registries signed by private certificate authorities yet.

## Roadmap

- Support for remote registries signed by private CA
- The ability to inspect templates and bundles for required variables
